**项目介绍：** Vue 实现一个在线多人博客分享的平台，实现有登录、注册、创建博客、修改博客、查看详情、删除博客等功能，用户在未登录状态下只有查看博客的权限，只有在登录状态下，才能拥有全部权限。

1､ **Vue CLI** 建模版 + **Axios** 取数据 +  **Vue Router** 切换路由、异步加载及权限验证 + **Vuex** 状态管理

2､ **Grid** 布局 +  **Less** 编译 + **Element UI** 交互 + 接口封装

**技术栈：** Vue CLI、 Vue2､ Axios、Vue Router、Vuex、ES6､ Npm、Grid、Less
